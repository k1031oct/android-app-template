# Development Plan

This document outlines the development plan for this project. It is organized into phases. Use the "Generate Issues from Plan" workflow to automatically create issues from this plan.

## How to Use

1.  Define your development phases below, following the `### フェーズX:` format.
2.  Under each phase, list the tasks using the `- [ ] **Task:** Your task description (frontend)` format.
    *   Include `(frontend)` for frontend-related tasks to automatically add the `frontend` label.
3.  Run the "Generate Issues from Plan" workflow from the Actions tab, specifying the phase number you want to generate issues for.
